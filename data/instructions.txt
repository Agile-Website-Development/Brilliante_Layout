1. Go to your Dashboard > Tools > Import > WordPress.

2. You'll be asked to install the WordPress Importer plugin, please go ahead and install it.

3. Click on "Activate Plugin & Run Importer".

4. Browse for the "content-data.xml" file and upload it.

5. On the next screen, you'll have the option to assign a new author's name. I created the content using the designer's name here. It's up to you on this one. No need to check attachments.

6. You could delete the "Hello world!" post and the "Sample Page" page generated by the WordPress installation.

7. Finally, go to Settings > Reading > Blog pages show at most: "3 Posts".
